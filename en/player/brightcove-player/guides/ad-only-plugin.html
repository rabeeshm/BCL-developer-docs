Ad Only Plugin

In this topic, you will learn how to use the Ad Only plugin to allow you to play IMA3 ads in a player with no normal video content.

Overview

Currently, to use the ad only plugin you need to set and then lock your player on version 5.8.1. This needs to be done using the Player Management API.

The ad only plugin also has a compatibility issue with the IMA3 plugin. If you wish to use the ad only plugin with IMA3 you need to be using version 2.11.0 of the IMA plugin. The JavaScript and CSS files for this plugins are located here:

  • //players.brightcove.net/videojs-ima3/2.11.0/videojs.ima3.min.js
  • //players.brightcove.net/videojs-ima3/2.11.0/videojs.ima3.min.css

The plugin can be added to the player in normal plugin adding ways using Studio (http://docs.brightcove.com/en/player/brightcove-player/guides/plugin-guide.html#Adding_plugins_in_Studio).

In Studio, you must add the IMA3 plugin manually, not using the ADVERTISING section. If you use the ADVERTISING section to add the IMA3 plugin, you will not be using a compatible version of the IMA3 plugin, as just mentioned. The steps below detail how to manually add the correct version of the IMA3 plugin.

You can also use the Player Management API (http://docs.brightcove.com/en/player/player-management/index.html).

The sample video below demonstrates the use of the Ad Only Plugin. The ad that plays is configured as a preroll in a VMAP/VAST XML document. In this case the player is NOT set to autoplay, which you would most likely want for an ad only player, but that is a setting you can change.

Implement using Players module

To implement the Ad Only Plugin using the Players module, follow these steps:

  1. In Studio, open the PLAYERS module and create a player, or locate a player to which you want to add the plugin.
  2. Click the link for the player to open the player properties.
  3. Locate the Plugins section and click Edit, then click the JavaScript section.
  4. For the JavaScript URL enter the location of the JavaScript for the correct version of the IMA3 plugin: 
    //players.brightcove.net/videojs-ima3/2.11.0/videojs.ima3.min.js
  5. Click the plus sign at the end of the URL and another form field will be added to supply another JavaScript file location.
  6. For the second JavaScript file location enter the following for the ad only plugin: 
    //players.brightcove.net/videojs-ima3-tech/1/videojs-ima3-tech.min.js
  7. Check to be sure the JavaScript block appears as follows:
  8. Click the CSS section.
  9. For the CSS URL, enter:
    //players.brightcove.net/videojs-ima3/2.11.0/videojs.ima3.min.css
  10. Click the Name, Options (JSON) section.
  11. For the Name, enter ima3.
  12. Enter the plugin options. The following options control ad playback and point to a single ad's VMAP information.
    {
  "useMediaCuePoints": false,
  "vpaidMode": "ENABLED",
  "hardTimeouts": true,
  "timeout": 5000,
  "requestMode": "onload",
  "adTechOrder": [
    "flash",
    "html5"
  ],
  "serverUrl": "//solutions.brightcove.com/bcls/brightcove-player/vmap/one-preroll-vmap.xml"
}
  13. Click Save and then Publish the player.

You can now use either the iframe or in-page embed player implementations to use the ad only player.
