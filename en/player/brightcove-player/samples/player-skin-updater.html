Brightcove Player Sample: Player Skin Updater

This sample uses the Player Management API to update a player to use the Player version 5 skin.

Note: do not use this on a 1.x player!

The app is contained in a Codepen below - you can fork the Codepen if you would like to have your own copy. All the code is contained in the Codepen except for a proxy which makes the requests to the Player Management API. The code for the proxy is shown below the app.
