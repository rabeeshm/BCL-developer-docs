Lightbox example

Clicking the thumbnails below will show the player in a pop-up and load the videos.

The player is initially inside an zero-size div with overflow set to hidden, rather than hide the div using display:none. When the lightbox is "opened" the div is set to it's correct size.

If hidden using display:none, the HTML player would be sized incorrectly, and the Flash player would not be loaded before the lightbox was shown (on Chrome).

Big Buck Bunny Trailer

Elephants Dream

CSS for lightbox:

#lightbox {
  position: absolute;
  top: 25%;
  left: 50%;
  margin-left:-240px;
  color: white;
  text-align: center;
  background-color: black;
  z-index:20;
  overflow: hidden;
}
.show {
  width: 480px;
  height: 290px;
}
.hide {
  width: 0;
  height: 0;
}
