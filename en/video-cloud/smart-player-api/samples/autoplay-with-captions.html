Smart Player API Samples: Autoplay with Captions On

This example shows how to autoplay a video and also automatically turn on captions, if there are any.

API resources used

This sample uses the following API resources

Modules and methods

  • VIDEO_PLAYER
    • play()
  • CAPTIONS
    • setCaptionsEnabled()

Events

  • templateLoad
  • templateReady

The player

A standard Chromeless Video Player is used for this sample.

Note: the buttons below are for development purposes only — do not use in production.

Switch to HTML5 Player Switch to Flash Player

How it's done

This one is very easy — all you need to do is get references to the Video Player and Captions modules, and then call one method of each. To turn captions on, you call the setCaptionsEnabled() method of the Captions module, and to start the video, you call the play() method of the Video Player module.
