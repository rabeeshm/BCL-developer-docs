Update Media

This sample uses the updateMedia() of the CONTENT module to modify properties of the current video in the player. The changes only affect the video DTO on the client, not the video in your Video Cloud media library. Here updateMedia is used to remove the video name that is automatically displayed in the upper left portion of the Chromeless Video Player overlay. The overlay itself cannot be modified through the API (or BEML), so the only way to remove this display is to remove displayName data.



Note: This will only work in single video players, NOT playlist players.

API resources used

This sample uses the following API resources

Modules and methods

  • VIDEO_PLAYER
    • getCurrentVideo()
    • play()
  • CONTENT
    • updateMedia()

Events

  • templateLoad
  • templateReady

The Player

A standard Chromeless Video Player is used for this sample.

How it is done

All you need to do here is call getCurrentVideo(), and then, in the callback, change properties in the videoDTO and call updateMedia(), passing the editing video DTO.

Note: updateMedia() only works before playback begins.
