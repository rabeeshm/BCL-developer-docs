Smart Player API Samples: Use Full Browser Window Width on Play

This example shows how to expand the Brightcove smart player to the full width of the browser window on play, while maintaining the aspect ratio.

For a more detailed discussion of getting started with the Smart Player API see the Getting Started Guide.
