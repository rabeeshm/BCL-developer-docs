Class Detail
brightcove.api.BrightcoveExperience()
Class to manage API interaction with embedded Brightcove player. This constructor should not be called directly by the developer, but is handled within the Brightcove code. To obtain a reference call: 
var experience = brightcove.api.getExperience(id);

{brightcove.api.modules.APIModule} getModule(moduleName)
Returns the API module based on the name.

The name of the module to return an instance of. A constant from APIModules should be used.
