Media API Sample: unshare_video

This sample uses the Media API unshare_video() WRITE method to unshare a video from one account to another. Note: sharing must be enabled between the two accounts, and the video must have been already shared for this to work.

There are hard-coded values from a Brightcove Learning demo account that you can use to test with, or you can substitute a write token and video ID from your own account, along with the ID for account you wish to unshare.

Note: you should never hard-code your tokens into a client-side application in the manner shown here for demonstration purposes only. Also be aware that even if you require the use to enter the token, it will be transmitted as plain text unless the form submits over an https connection.
