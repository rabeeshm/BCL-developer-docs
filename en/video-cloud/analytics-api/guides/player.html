Date Dimension Reports

The report supports date dimensions. The user can specify the from and to parameters in yyyy-mm-dd format. This enables a user to query the analytics system to generate report report by days within the from and to date range. Any date range specified by the user will be interpreted to the current timezone for the user. The date ranges for the request are inclusive, exclusive i.e. if the user makes a request from=2013-09-24&to=2013-09-27, we will return results from 2013-09-24 00:00:00 to 2013-09-27 00:00:00.
