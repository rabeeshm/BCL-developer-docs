referrer_domain Dimension

Overview

The referrer_domain dimension is the top private domain of the referrer URL on a page where events are collected for activity metrics (this URL is sent to Collector with the source parameter). The referrer_domain dimension only includes metrics from player instrumentation events.

referrer_domain in the API

Referrer URL API Value

http://blog.brightcove.com/en/

(any URL with a valid public domain)

brightcove.com

(top private domain for the URL)

http://localhost

http://192.168.0.0

(any private domain or IP address)

non_public_domain

(direct traffic, invalid URL)

null

Report Data

Report data can be broken down by referrer_domain. In addition, report data broken down by referrer_domain can be filtered for a specific video or player using the where parameter.
