Samples: Video Engagement Graph

This sample allows you to generate an engagement graph for a video in your account. Although you can generate an engagement graph from a video report, here we use a engagementrequest, because the response contains more complete data: rollups return engagement for every hundredth segment of the video, while a report returns only engagement at each quarter point of the video.
