Dynamic Ingest Error Message Reference

This is a reference to error messages returned by the Dynamic Ingest API. Also see the error message reference for the CMS API, which is used to create the video object before the video is ingested.

Error messages

The follow error messages can be returned by the Dynamic Ingest API

Name Message Cause
NOT_SUBMITTED Unable to submit job, please try again later. System issue - unable to create the job in the ingest system
INTERNAL_ERROR Internal error, please try again later. Issue with API
NO_SUCH_VIDEO Unable to find the referenced video. Bad URL for the video source file.
UNAUTHORIZED Unable to authorize request. Missing or bad OAuth credentials.
BAD_ACCOUNT Account ID was missing or invalid. Invalid account ID specified.
NO_SOURCE Unable to find a source to use. Unable to find a usable video file at the address specified.
PROFILE Unable to find profile by name. Could not find the specified ingest profile in the account
CDN_CREDENTIALS Unable to fetch CDN credentials Could not get the credentials for the account CDN
BAD_CALLBACKS Callbacks were not in expected format Callback URLs were not specified correctly.
SUBMISSION_FAILURE Unable to submit job please try again later. Unable to submit job please try again later
AUTH_ERROR Unable to authorize request. OAuth service unavailable
BAD_VIDEO Video ID was missing or invalid. Video id was not included or invalid
MALFORMED_SOURCE_URL source url is malformed. Source URL was incorrect or incorrectly encoded
BAD_PROTOCOL_SOURCE_URL source url uses a unsupported protocol. Only HTTP, HTTPS, or S3 are allowed
EXCEED_MAXIMUM_VTT_SOURCES vtt sources exceed the maximum size. vtt sources exceed the maximum size.
INVALID_VTT_SRC_LANGUAGE vtt srlang isn't a ISO639 alpha2 langurage code. vtt srlang isn't a ISO639 alpha2 langurage code.
INVALID_VTT_KIND vtt kind is invalid. check the valid kinds for webVTT files.
NO_CAPTURE_IMAGE_ALLOWED capture-image is not allowed if an image source is provided. capture-image is not allowed if an image source is provided.
UNPROCESSABLE_ENTITY request data contains some unprocessable entity. request data contains some unprocessable entity.
MALFORMED_REQUEST Unable to parse request body. Unable to parse request body.
CDN_CONFIGS Unable to fetch CDN credentials Unable to fetch CDN credentials<
AMBIGUOUS_REQUEST Both a master url and use_archived_master were set in the request. Both a master url and use_archived_master were set in the request.
