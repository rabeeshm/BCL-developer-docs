Property detail
adTranslatorproperty
public var adTranslator:AdTranslation

A reference to the ad translator instance to use as the current translator. If this is set prior to AdRules initialization, the specified ad translator will be set as the player's current translator. Otherwise, it can also be set by using AdRules.setAdTranslator().

adTranslatorURLproperty 
public var adTranslatorURL:String

The URL of the ad translator that should be loaded. If this is set prior to AdRules initialization, the specified ad translator will be loaded into the player. Otherwise, it can also be loaded by using AdRules.loadAdTranslator().

requestKeyValuePairsproperty 
public var requestKeyValuePairs:String

Holds the additional key value pairs for an ad request made through AdRules.callAdTranslator.

useDefaultTranslatorproperty 
public var useDefaultTranslator:Boolean

A Boolean specifying whether or not to use the default Brightcove DFP translator as the current translator. If this is set prior to AdRules initialization, the DFP translator will be set as the player's current translator. Otherwise, it can also be set by using AdRules.useDefaultDFPTranslator().
