Sample XML Manifest for Batch Provisioning

This sample XML Manifest file uploads a video file, thumbnail image, and still image, links these assets together to create a video, and creates a manual playlist, using the Video Cloud FTP Batch Provisioning feature. These are the most common tasks you might perform with FTP Batch Provisioning.

The most common usage of FTP Batch Provisioning is for integrating your Content Managment System (CMS) with Video Cloud for video publishing. Or, you might use FTP Batch Provisioning to ingest a large amount of assets (videos and playlists) on the Video Cloud service. The XML manifest file sample included in this document is the simplest example of the actions you can perform with the FTP Batch Provisioning process. For a more complete understanding of actions you can perform with FTP Batch Provisioning, see the following documents:

  • Using FTP Batch Provisioning
    This document provides an overview and the steps to follow to perform the FTP Batch Provisioning process. 
  • FTP Batch Provisioning: Reference for the XML Manifest
    This reference defines the XML syntax for the XML tags and attributes for the actions you want your XML manifest file to perform once it is uploaded to the Video Cloud server. It also links to the Document Type Definition (DTD) for validating your XML manifest file. 
  • FTP Batch Provisioning: Troubleshooting
    This document explains behavior and error messages you might encounter and how to address those issues.
