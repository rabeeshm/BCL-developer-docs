Brightcove Player SDK for tvOS

Welcome to Brightcove's Player SDK for tvOS developer documentation. Here you will find information for playing video content on Apple TV:

  • Video Cloud customers can play Video Cloud library content
  • Perform customers can play videos with internet accessible URLs

Supported versions

In order to take advantage of the latest technology improvements from Apple, we currently offer support for:

  • tvOS 9

Getting started

Before beginning development with the Player SDK for tvOS, there are a few concepts you should learn about:

  1. Media

    The current version of the Player SDK for tvOS supports playing content from your Video Cloud account, either through the Media API or the Playback API. It also supports playing content with internet accessible URLs.

  2. Analytics

    With the Player SDK for tvOS, Video Cloud customers automatically get robust analytics data reported in Video Cloud.

  3. SDKs developer forum

    Join us in the SDK developer forums where SDK related questions are answered by the community and Brightcove engineers. Currently a private group, you'll have to request access, providing details about the project you're working on to gain entry.

Download the SDK

The Brightcove Player SDK for tvOS is contained within the Player SDK for iOS. Here, you will find two installation packages:

  • A static library framework which supports iOS 7
  • A dynamic framework which supports the following:
    • iOS 8 and higher
    • tvOS 9.0 and higher

Get the latest version of the Brightcove Player SDK for iOS [5.0.2] along with the following:

  • The SDK podspec
  • Documentation
  • Release Notes
