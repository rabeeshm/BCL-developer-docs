Schema Builder

This app uses the Playback API to generate a VideoObject Schema based on the schema.org specification. Including a schema with your video provides information to search engine indexers and enhances SEO.

You can generate the code here and use it as your publishing code if you are using a single Brightcove player for the account and a standard (iframe) embed. If you are using another player or the advanced (in-page) embed, you can just use the schema part of the generated code and insert it above your player embed code in the page.

Note: this is a client-side app, and you can make requests to the Playback API from the client-side. However, you cannot simply use this script in pages with your players to generate effective schemas, because the schema code must be injected into the page on the server side, before the page is delivered to the browser, to be recognized by search engines. If you are serving your pages via a web server application, use the logic here to get the information using either the Playback or CMS API and embed it in the page before it is served.

Also note that this app produces code for a single video player only, and if you are embedding the schema code manually, you should update it periodically, as URLs can change
