Getting Started with IE8

NOTE: IE8 and IE9 are no longer supported by Microsoft and make up a tiny fraction of Brightcove's traffic, and are therefore no longer actively supported. Brightcove will consider fixing issues related to these browsers on a case-by-case basis.

This topic discusses how you can run the Brightcove Player In-Page embed code in the IE8 browser.

Since the IE8 browser was created before the invention of HTML5 tags, the browser does not know about these tags. Ideally, Microsoft would like users to upgrade to IE9, but this topic will show you a few tips to get the HTML5 video element to play in IE8.

Note: The extra setup steps in this topic are only needed if you are using the Brightcove Player In-Page implementation with IE8. If you are using the iframe implementation, you can skip this topic.
