Use in HTML

If you are using the URLs for the plugins directly in an HTML page, you would use a <script> tag for the JavaScript and a <link> tag for the CSS. Examples follow:

<script src="//players.brightcove.net/videojs-ima3/2/videojs.ima3.min.js"></script>
<link href="//players.brightcove.net/videojs-ima3/2/videojs.ima3.min.css" rel="stylesheet">
