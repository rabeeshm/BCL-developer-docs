Notes

  • [1] Latest

  • [14] Support for Flash ads on IE is more robust with IMA3
  • [15] Video Cloud-hosted content is available via HTTPS when necessary. If you are using your own CDN, you must provide HTTPS URLs for all content. See HTTPS Support Overview document for more information.
  • [16] Live UI not applied
  • [17] Planned for first half of 2015
  • [18] Dependent on Facebook allowing HTML5 video.
  • [19] MP4 only - pending resolution of HLS/SSL issue
  • [20] Versions 4.x and newer
  • [21] Versions 7, 8, 9
  • [22] Note that DRM for videos viewed in browsers is supported for desktop only, not device mobile web browsers. Widevine modular and PlayReady are supported for MPEG-DASH. For HLS content DRM is not currently available, but you can use HLS encryption (HLSe) to provide a lesser degree of content security.
