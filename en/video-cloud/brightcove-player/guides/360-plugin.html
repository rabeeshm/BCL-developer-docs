360º Video Plugin

Note: The 360º Video Plugin is in beta development. The 360º plugin can be used on production web sites where the following limitations are acceptable:
  • The plugin only works on the following desktop browsers: Chrome, Firefox and Edge (no mobile support).
  • Players with this plugin will treat all videos as 360º videos.

This plugin turns a properly recorded video element into a HTML5, panoramic, 360º video that can be played in Brightcove Player.

The sample video below demonstrates the use of the 360º Video Plugin. At this time Chrome, MS Edge and Firefox support the plugin. A Brightcove Player error will be display if used in a non-compatible environment. Once you click on the video to play it, you can use either your mouse or the arrow keys to control the view. The 360 video logo play button fades on play.

Implementing the plugin is straightforward, just complete these two steps:

  • Properly ingest the video.
  • Add the plugin to the player.

Ingest the 360 video

Once you have the 360 video recorded, you need to ingest it into Video Cloud. A little understanding of how 360 video works will help you understand some reasons for the Ingest Profile to choose. The 360 video is filmed to be projected on the interior of a sphere. This means that the visible area of the video in the player is only a fraction of the entire video. The following screenshot gives you a rough idea of this concept. The area inside the red rectangle is what is visible in the player, but only a small part of the entire video.

Since only a small part of the video will be in the player at any one time, you want a very high quality rendition to be playing, with no chance of stepping down to a lower quality rendition. In essence, you are "zooming in" on part of the video and a poor quality rendition would yield a very poor quality viewing experience. For this reason, during the ingestion process you should use a Ingest Profile of high-quality, and few renditions. Brightcove recommends using the single-bitrate-high (2 renditions) Ingest Profile for 360 videos.

Implement using Players module

To implement the 360º Video Plugin you will add a JavaScript link and a CSS link. You will then add two plugins that use those files. Follow these steps:

  1. Open the PLAYERS module and either create a new player or locate the player to which you wish to add the plugin.
  2. Click the link for the player to open the player's properties.
  3. Locate the PLUGINS section and click Edit.
  4. The JavaScript for the plugin is provided and will not change for different player implementations. For the JavaScript URL, enter: 
     //players.brightcove.net/videojs-vr/beta/videojs-vr.min.js

    NOTE: This URL will change at times during beta development, so check back for the latest, correct value.

  5. The CSS for the plugin is provided and will not change for different player implementations. For the CSS URL, enter:
     //players.brightcove.net/videojs-vr/beta/videojs-vr.css

    NOTE: This URL will change at times during beta development, so check back for the latest, correct value.

  6. For the first plugin function, enter vr for the name.
  7. Enter the corresponding plugin options.
    {
  "projection": "Sphere"
}
  8. For the second plugin function, enter excludeBrowsers for the name. No options need be entered.
  9. Click Save in the the PLUGINS section.
  10. Locate the STYLING section and click Edit.
  11. For the optimal viewing experience, it is best to set the Enable Title and Description Dock to Disabled.
  12. Click Save and then Publish the player.
  13. Select the MEDIA module.
  14. Publish your 360 video using your newly altered player that uses the 360º Video Plugin.
