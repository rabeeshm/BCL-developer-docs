Quick Start to Using Ads

This document provides a hands-on introduction to using the IMA3 Plugin to display advertisements in Brightcove Player.

Verify VAST tag

To be sure the VAST ad tag you have is valid with Google follow these steps.

  1. Obtain an ad tag. (If you do not have one, use the one that follows this step for demo purposes.) In this quick start it is assumed you have an ad that includes a pre-roll, but any inline ad will work.

  2. To verify the validity of your ad tag, be sure you have copied its URL and browse to the following page:

    Video Suite Inspector (clicking this link will cause the page to open in a new window or tab)

  3. Paste your ad tag URL into the Input type form input field.
  4. Click Test Ad and you will see your ad play, followed by the Google supplied motorcycle race video.
  5. Be sure the test complete successfully, then proceed to the next section. If the test failed, you need to check your ad tag.

Note: The VAST ad tag can be hosted by your ad provider, or you can copy the XML and host it on your own URL.

Create player

To create a new player, follow these steps.

  1. Login to Perform Studio at //studio.brightcove.com/products/perform/players/.
  2. Login to Video Cloud Studio at //studio.brightcove.com/products/videocloud/players/.
  3. The current account name will appear in the upper right corner of the page. If you have multiple accounts, click the current user - current account link and select the account in which you would like to create the player.
  4. Click the Players link in the navigation header.
  5. Click the New Player button.
  6. Enter a Name and Short Description for the player. For this Quick Start, the player name will be Sample Player.
  7. Confirm that the newly created player appears in the list of players.

Add plugin to player

Brightcove Player relies on the IMA3 Plugin to play VAST ads. To add said plugin to the player just created, follow these steps.

  1. Click the link for Sample Player to open the player properties page.
  2. Locate the Plugins section and click Edit.
  3. Click in the JavaScript row and paste in the following URL which points to the JavaScript code for the IMA Plugin.

    http://players.brightcove.net/videojs-ima3/videojs.ima3.min.js

  4. Click in the CSS row and paste in the following URL which points to the stylesheet for the IMA Plugin.

    http://players.brightcove.net/videojs-ima3/videojs.ima3.min.css

  5. Click in the Name, Options (JSON) row and enter ima3 for the plugin name.
  6. For the options, paste in the following JSON code, then enter your ad tag in the now empty quotes for serverUrl. 
    {
  "serverUrl": ""
}
  7. Ensure the Name, Options (JSON) appears as follows, except your ad tag is used.
  8. Click Save and you will see you have 1 configured value for each option.
  9. Follow the process to publish your player.

Publish video in player

You publish a video in the newly created player in the Media module. Once you associate the video with your player you publish the video and will then have access to three implementations in which to view the video in your player.

To publish a video your player, follow these steps:

  1. In Video CloudPerform Studio navigate to the Media module by clicking on the Media link in the main menu.
  2. Click on the thumbnail for the video you wish to publish in your player, then click the Publish option.
  3. From the select control in the Select a Player section of the page, choose your newly created player.
  4. You now have access to the three implementations of the player, iframe, In-page and URL. Do not close this window as you will be copying code from it.

Note: If curious, you can see "under the hood" of the player configuration using the URL code. Click on the URL code link, then copy and browse the generated URL. Next, change the index.html at the end of the URL to config.json. You will see a JSON object detailing the current configuration of the player you just created.

Assign video to player

The Video Content properties are used to assign video content to the player.

To configure the video content properties, follow these steps.

  1. Locate the Video Content section and click Edit.
  2. Add the rendition or renditions of your choice.
  3. Add a poster image. If you do not add a poster image the first frame of the video will be used.
  4. Do not set any dimensions for the video as this will be done via CSS in the page.
  5. Click Save.
  6. Follow the process to publish your player.

For more information on the Video Content properties, see Configuring the Video Content Properties for a Player.

Test

Of course, you will want to be sure the ad is working in your player.

To see your add playing in your player, follow these steps.

  1. From the three implementations click the URL option.
  2. Copy the displayed URL.
  3. Paste this URL into a new window or tab and browse it.
  4. Play the video and you will see your ad play, followed by the video you assigned to the player.

Test

Of course, you will want to be sure the ad is working in your player. In this case you will test using the preview version of the player so there is no need to wait for the player to actually be published. Although the publish time is not long, using the preview version of the player eliminates any confusion of something not working because the player has not completed publishing.

To see your add playing in your player, follow these steps.

  1. Near the top of the page select Embed Code & URL then Preview Player.
  2. Copy the Player URL.
  3. Paste this URL into a new window or tab and browse it.
  4. Play the video and you will see your ad play, followed by the video you assigned to the player.
