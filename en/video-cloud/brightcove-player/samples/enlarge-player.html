Enlarge Player on Mouseover

The page you are looking for doesn't live here anymore -- redirecting you to in 3 seconds.

In this topic, you will learn how to use JavaScript and CSS to enlarge a small player when you mouse over it, and then shrink it to small size when you mouse off.

Implementation details

There is additional logic using the Player API to remove the shrink behavior while the video is playing, and then restore it again when the video is paused or ends. As created here, this sample must use the Advanced (in-page) player embed. You could create this functionality using the iframe embed code also, but the logic would be a little different and would not lend itself to being turned into a plugin.

How it works

The following sections explain the player, JavaScript and CSS code used in the example. We will build the code to all work in the page initially for testing, but we're create it in a way that we can add it to the player as a plugin once we know it's working correctly.

The HTML code

  1. Add the following HTML code to your page code wherever you want the player to appear initially: 
    <div id="playerWrapper">
</div>

Create the player

The Brightcove Default Player was used for this sample. To create a new player, and assign a video to it, follow these steps:

  1. In the Players module, duplicate the Brightcove Default Player.
  2. Publish the player.
  3. In the Media module, select a video and publish it with this player.Assign video content to the player and publish it. Set the width and height of the player to 100%.
  4. Copy the advanced embed code and paste it into the page.
  5. Add the id inpagePlayer to the video tag.
  6. Wrap the entire player embed code inside a div tag that has the id playerWrapper.
  7. The whole HTML for the player and wrapper should look like this: 
    <div id="playerWrapper">
    <video id="inpagePlayer" data-video-id="4825279542001"
    data-account="1752604059001"
    data-player="4kHAiDICl"
    data-embed="default"
    class="video-js"
    controls
    style="width: 100%; height: 100%;"></video>
    <script src="//players.brightcove.net/1752604059001/4kHAiDICl_default/index.min.js"></script>
</div>

The CSS code

Next we'll add a little bit of CSS code to provide initial styles for the player wrapper. This code could go in the HTML page, but we'll create it as a separate CSS file so that we can add it to the player later.

  1. Create a new file called enlarge-player.css - you will need to put this file on a public web server later so that Video Cloud can see it.
  2. Add the following code to the CSS file: 
    /* initial styles for the player wrapper */
#playerWrapper {
    width: 272px;
    height: 153px;
    transition: width .4s ease-in-out, height .4s ease-in-out;
}
  3. Now add a link tag to the head section of the HTML page to include the CSS file: 
    <link rel="stylesheet" href="path-to-the-css-file/enlarge-player.css">
  4. Note that the style rule simply sets the height and width of the wrapper element, and adds a simple transition for the width adn height to provide a little animation when we resize it.

The JavaScript code

Now we'll add JavaScript to handle the resizing of the player wrapper. Like the CSS code, this could go in the HTML page, but we'll create it as a separate file to add to player as a plugin later.

  1. Create a new file and save it as enlargePlayer.js - like the CSS file, you will need to put this on a public web server later.
  2. Add the following code to the file: 
    videojs.plugin('enlargePlayer', function() {
    var player = this,
        // set the large and small sizes
        // you could also all options to be passed in to set these sizes
        largeWidth = 640,
        largeHeight = 360,
        smallWidth = 272,
        smallHeight = 153,
        // get a reference to the player wrapper
        // if that element doesn't exist in the HTML, this plugin will fail
        playerWrapper = document.getElementById('playerWrapper');

    /**
     * make the player large
     */
    function makePlayerLarge() {
        playerWrapper.setAttribute('style', 'width:' + largeWidth + 'px;height:' + largeHeight + 'px;transition: width .4s ease-in-out, height .4s ease-in-out;');
    }

    /**
     * make the player small
     */
    function makePlayerSmall() {
        playerWrapper.setAttribute('style', 'width:' + smallWidth + 'px;height:' + smallHeight + 'px;transition: width .4s ease-in-out, height .4s ease-in-out;');
    }

    /**
     * add the mouseout event listener
     */
    function mouseoutOn() {
        // add the event listener to make the player small on mouseout
        playerWrapper.addEventListener('mouseout', makePlayerSmall);
        // make the player small now
        makePlayerSmall();
    }

    /**
     * remove the mouseout event listener
     */
    function mouseoutOff() {
        // remove the mouseout event listener
        playerWrapper.removeEventListener('mouseout', makePlayerSmall);
    }

    // make sure the player wrapper exists - otherwise, do nothing
    if (playerWrapper !== null) {
        // initially, make sure the player is small
        makePlayerSmall();
        // on mouseover make the player large
        playerWrapper.addEventListener('mouseover', makePlayerLarge);
        // on mouseout, make the player small
        mouseoutOn();
        // on play, keep the player large
        player.on('play', mouseoutOff);
        // on pause or video end, make the player small
        player.on('pause', mouseoutOn);
        player.on('ended', mouseoutOn);
    }

});
  3. Add the following code to your page, somewhere after the player code, to include the JavaScript file and activate the plugin: 
    <script src="./assets/enlarge-player/enlarge-player.js"></script>
<script>
    videojs('inpagePlayer').enlargePlayer();
</script>
  4. Save changes and browse the page through your local or remote web server - it should function like the sample above. The comments in the JavaScript explain how it works.

Make a plugin

If your code is working correctly, you can now add it to the player as a plugin.

  1. Move your plugin JavaScript and CSS files to an Internet accessible location of your choice.
  2. Go back to Studio and locate your player.
  3. Locate the PLUGINS section, and click the Edit button.
  4. In the JavaScript area, enter the path to your Internet accessible JavaScript file.
  5. In the CSS area, enter the path to your Internet accessible CSS file.
  6. In the Name, Options (JSON) area, enter the plugin name (enlargePlayer) and an emptry JSON object {} since there are no options.
  7. Save and publish your player.
  8. You can now remove the link tag you added to your HTML page in step 11, and the script tags you added in step 14, as they are no longer needed.
  9. Save and test your page again - it should work as before.

Implementation notes

  • If you want to be able to specify the small and large sizes of the player for each instance, you could pass those in as options to the plugin, and then modify the plugin code to read the values from the options object. See the Quick Start to Plugin Development for more detail on how to do this.
  • If you want to use the iframe player instead of the inpage player. The player wrapper div would surround the iframe tag, and the Javascript to manage the size of the player would be very similar to what you see here. It would need to be in the HTML page, however, rather than a player plugin.
