This document is an overview of the dimensions collected for the video_cloud domain, and the metrics exposed by the API. More information for a particular dimension can be found in the specific detailed documentation.

Dimensions: video_cloud

The video_cloud domain collects data for the following dimensions:

Dimension

Explanation

account

Video Cloud account id.

player

Video Cloud player id (belonging to the account).

video

Video Cloud video id (belonging to the account).

referrer_domain

Top private domain of the Referer Header on a page where player/video events are collected
