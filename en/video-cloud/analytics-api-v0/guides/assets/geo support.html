Geography dimensions in the API

Location detection is done using Max Mind GeooIP2 city dataset.



Dimensions

  • country - As the ISO-3611-1 country code. eg: 'US'
  • region - As the ISO-3611-2 region code. eg: 'US-WA'
  • city - City name. eg: Seattle

Note: For unknown country or region the API returns "ZZ" as the code (As per ISO-3611-alpha2).



Metadata

  • dma - Designated Market Area (DMA) code for the location (Only available for US locations).
  • latlng - Latitude and Longitude of the city center.



Example

Request:
https://data.brightcove.com/analytics-api/videocloud/accounts/1234/report?dimensions=city&where=country==US
