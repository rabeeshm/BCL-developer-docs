Fields Parameter

The fields parameter is an optional parameter that allows the client to include/exclude items in the API response.

  • The API response would generally always contain the fields mentioned as the dimension, sort and fields parameter.
  • In case the fields parameter is not specified then we would consider it to be video_view.
  • The other metrics available are specific to the combination of dimensions provided and may be restricted by using different values of the fields parameter as a comma-separated list.


Examples

--- Request ---

--- Description --

/videocloud/accounts/1234/report?dimensions=video

The report will only have the video and video_view for the different items

/videocloud/accounts/1234/report?dimensions=video&sort=player_load

The report will only have video, video_view and player_load for different items.

/videocloud/accounts/1234/report?dimensions=video&fields=video_impression

The report will only have video and video_impression for the different items.

/videocloud/accounts/1234/report?dimensions=video&fields=video_view,video_impression

The report will only have video, video_view and video_impression for the different items.

/videocloud/accounts/1234

The rollup will only have the video_view in the metrics item of the response

/videocloud/accounts/1234&fields=video_impression&sort=video_view&fields=active_media

The rollup will have the video_view, video_impression and active_media in the metrics item of the response
