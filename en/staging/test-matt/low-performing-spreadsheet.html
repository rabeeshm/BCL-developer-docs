Identify and Delete Low-Performing Videos

Introduction

For a number of reasons, it is probably best to not have videos in your catalog that are not viewed, and you guess they won't be in the future. In this document you will see how to identify these videos and remove them from your library. The following bullet points provide a high level overview of the process. The detailed steps then follow.

  • Use the Analytics module to create a custom report that includes the video ID, title, video impressions and video views. The report will be exported in a comma separated values (CSV) format.
  • Import the CSV file into a Google Docs spreadsheet.
  • Sort the videos by the video_view property, low to high.
  • Determine the videos you feel meet your criteria for "low performing" and are wishing to delete from the catalog.
  • Copy the IDs of the low performing videos to a Google Doc and manipulate the format to use in a bulk deletion process.
Deleting videos is a drastic operation and it is not possible to undelete them. For safety's sake, be sure you have a local copy of a video before deleting it.

Export all videos data

From the Media module you can get a report on all media in your catalog. This will be used later in a spreadsheet.

  1. Enter Studio's Media module.
  2. Click on the Export button near the top of the page.
  3. In the dialog that opens, be sure All videos in the list is selected.
  4. Click the Export Data button.
  5. When the report has finished building, click the Download File button.
  6. Note that the file name corresponds to your account ID.

Create custom Analytics report

To create the CSV report perform the following steps:

  1. In Studio, move into the ANALYTICS module.
  2. On the left side menu, click Custom.
  3. Use the Include Accounts dropdown to choose the desired accounts for the report.
  4. Choose the Report Type to be One-time.
  5. Select a Date Range for the report using the calendars.
  6. For the Data Groups choose Video.
  7. For the Data Fields choose the three fields Video Name, Video Views and Video Impressions:
  8. Below the fields you see a preview of your report. Note that the video ID is automatically included in the report:
  9. Supply a Report Name or accept the default.
  10. Choose the Format to be CSV.
  11. Choose to get an email or not when the report is ready. Mostly likely you will just wait for the report unless your account contains a very large number of videos.
  12. Click the Request Report button.
  13. After some delay, you will see the report is downloaded.

Import and sort the data

In this document Google Docs will be used because the functionality is available to all for free. If you are an Excel user, you can use that to perform all the spreadsheet tasks shown in this document.

In this section of the document you will open the custom report just created and then sort to group the lowest performing videos.

  1. Open a Google Docs blank spreadsheet.
  2. Select File > Import > Upload then either drag the CSV file to the proper location or browse to select it.
  3. On the Import file dialog, accept the defaults and click Import data:
  4. Your imported data will appear similar to the following:
  5. Shift-click on the uppercase letter at the top of each column. You will see the data in the spreadsheet is selected.
  6. From the spreadsheet menu, select Data > Sort range....
  7. Check the Data has header row checkbox.
  8. From the Sort by dropdown select video_view. The dialog should appear as follows:
    Most customers will want to sort on video_ view. Although your business rules may make video_impression a better data field on which to sort. As a reminder, here are the definitions for both fields:
    • Video View: The number of times a video started playing, recorded when the stream is started (does not include rewind or replay).
    • Video Impression: The number of times a video was loaded into a player and ready for interaction.
  9. Click the Sort button.

Determine low-performing videos and delete them

You now need to determine how "low-performing" is defined for your organization. This then leads you to the number of videos to delete.

  1. Highlight, then copy, ONLY the video IDs for the videos you wish to delete. Do not include any videos that are missing an ID or name as this means the video is all ready deleted.
  2. You now have two options on how to proceed. They are:
    • If you have a small number of videos to delete, you can simply go into Studio and manually delete in the MEDIA module.
    • If you have a significant number of videos to delete, you may wish to perform the following OPTIONAL steps to ease the burden of deleting the videos.
  3. Open a Google Doc document.
  4. From the document menu, select Edit > Paste without formatting. You will simply see the video IDs in a column. To bulk erase the videos they must be in the format id:12345678 id:23456789 .... Rather than editing the list by hand, continue with these steps.
  5. Select Edit > Find and replace...
  6. For the Find value enter \n. This represents a carriage return.
  7. For the Replace with value enter a space followed by id:. The space in front of the id: is crucial for correct formatting.
  8. Check the Match using regular expressions checkbox.
  9. Click the Replace all button, then close the dialog.
  10. Check to be sure your list of video IDs now appears similarly formatted to following:
