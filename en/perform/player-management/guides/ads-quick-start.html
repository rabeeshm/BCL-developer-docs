Quick Start to Using IMA Ads

To utilize ads with the Brightcove Player there are a number of tasks to perform. The first is establishing a relationship with an advertisement provider, such as Google or FreeWheel.

Once you have an ad provider, you can proceed to utilize those ads with the Brightcove Player. At a high level, you must perform the following to use ads in the Brightcove Player:

  • Confirm the ad tag is valid
  • Load the IMA3 plugin into the Brightcove Player
  • Provide configuration options to the plugin, either at player initialization or dynamically

Validate ad tag

If your ad tag is not functioning correctly, no matter how good a job you do integrating it with the Brightcove Player it still won't work. For this reason, it is always good to confirm the validity of your ad tag before using it, or if problems should occur after usage has begun. For IMA ad tags, Google provides the Video Suite Inspector to check whether your video ad response will work. To test, go to the inspector and perform the following steps:

  • Choose an Input type
  • Paste the ad tag into the input box
  • Choose the Preferred ad type
  • Click Test Ad

If the ad tag is valid, you will begin to see the ad(s) playing in the Master Ad player.

Note: For testing purposes, you can go to the Video Suite Inspector and click the Sample link, and a functioning ad tag will be inserted into the input box. You can then copy the URL provided and use that for an example ad tag.

Use plugin

Note: The process shown in this section of the document, including and calling the the plugin via JavaScript, is for development purposes and useful for initial configuration and testing, but not considered a best practice. The best practice is to include the IMA plugin as part of your player configuration using a curl statement (as shown below) even if you're using the embed_in_page implementation. This way, you will automatically receive updates to the plugin permitting you to manage your ad configuration through the Studio, instead of having to modify each embed location directly.

Available ad framework

If you are a Brightcove partner working on your own ad library, or a customer wishing for a deeper insight into the ad utilization process, a framework is available to assist. The videojs-contrib-ads GitHub repository contains a plugin that provides common functionality needed by video advertisement libraries working with video.js.
