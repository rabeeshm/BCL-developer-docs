Display Thumbnail Previews Plugin

The thumbnail previews plugin allows for the timeline to display a preview image at defined points for your video. The user can mouse over the timeline and images defined by you will be displayed for specified time ranges.

Collect thumbnail images

At this time you must collect the images yourself. Preview thumbnails can be generated as part of the Zencoder encoding process. Also, tools exist like SnapMotion to automate this process.

No matter how you generate the images, you need to decide the number of images you wish to use and choose an appropriate size.

Implement using Players module

Thumbnails can be configured by using the thumbnails plugin. To implement the thumbnails plugin using the Players module, follow these steps:

  1. Open the Players module and locate the player.
  2. Click the link for the player to open the player properties.
  3. Locate the Plugins section and click Edit.
  4. The JavaScript for the plugin is provided and will not change for different plugin implementations. For the JavaScript URL, enter: 
     https://cdn.rawgit.com/brightcove/videojs-thumbnails/master/videojs.thumbnails.js
  5. The CSS for the plugin is provided and will not change for different plugin implementations. For the CSS URL, enter:
     https://cdn.rawgit.com/brightcove/videojs-thumbnails/master/videojs.thumbnails.css

Notice that the JavaScript and CSS files above use the RawGit CDN server. RawGit is a free provider that serves raw files directly from GitHub. While you can use this for testing, it does not guarantee 100% uptime for production.

For production use, we recommend downloading the JavaScript and CSS files for the Thumbnails Plugin from GitHub and putting them on your own server. Make sure your files are up-to-date before publishing your player.

This approach is a bit different from the other Brightcove plugins, because here we are using the RawGit server.


  1. For the Name, enter thumbnails.

  2. Enter the plugin options. The example below will display a thumbnail preview image as the mouse hovers over the timeline as follows:

    • The sea-lionfish-thumbnail.png image is be displayed from video start to 29 seconds
    • The sea-anemone-thumbnail.png image is displayed from 29 seconds to 54 seconds
    • The sea-clownfish-thumbnail.png image is displayed from 54 seconds to 84 seconds

      . . .

    • The sea-dolphins-thumbnail.png image is displayed from 143 seconds to the end of the video

    For details about the plugin options, see the Properties section.

  3. Click Save and then Publish the player.
  4. In the Media module, select a video and publish it with the player you have just updated with the Thumbnails Plugin.

Properties

The following plugin properties are used to control the thumbnail images:

  • first argument:
    • The value can be a string or number, and is associated with an object.
    • This defines the time in seconds you wish to display your thumbnail image.
    • At a minimum, you need a 0 property with a src value for when users hover over the beginning of the progress bar. If you have additional times, the plugin will partition the progress bar and display the appropriate image as the user hovers over the progress bar.

    Note: You need to include a 0 property with a src value. Otherwise, the plugin will try to use the example thumbnail and you may get an error.

  • src:
    • The value must be a string.
    • This is the URL for your thumbnail image.
