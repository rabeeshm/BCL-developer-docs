Set up cURL

cURL is a command line tool for making HTTP requests. We use cURL in a shell script for testing Brightcove API calls. If you are on Mac OS X or any Unix/Linux system, you already have cURL. If you are on Windows, see instructions below for getting cURL.

Windows

If you have Windows, you can download and install cURL as follows.

Installing cURL

  1. In Windows, create a new folder called curl in your C: drive.

    C:\curl

  2. Go to http://curl.haxx.se/download.html and download one of the following zip files:

    • If you have a Windows 64 system, scroll to the Win64 - Generic section and look for the latest Win64 ia64 zip version with SSL support. It's normally second in the list. Click the version number to start the download.

    • If you have a Windows 32 system, scroll to the Win32 - Generic section and look for the latest Win32 zip version with SSL support. It's normally second in the list. Click the version number to start the download.

  3. Unzip the downloaded file and move the curl.exe file to your C:\curl folder.

Sending Data

If you need to send data with the API request (as in a POST or PATCH request, for example), there are several ways to do it. If you need to submit the data as a form field, use the --form (or -F) flag, and then include the data inline or in a separate file. Here is a Media API example:

$ curl -F json=\<afl.json https://api.brightcove.com/services/post

Note: attach the file as json=\<afl.json, not json=\@afl.json - the "@" attaches the file as an upload, while the "<" just sends the file contents.
