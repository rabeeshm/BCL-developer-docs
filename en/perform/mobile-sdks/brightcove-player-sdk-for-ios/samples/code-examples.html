Limit the bitrate

You can't control which rendition in the HLS manifest gets selected by the AVPlayer, but you can put a bitrate cap on playback. This prevents the player from using renditions with a bitrates over the specified bitrate.

Set the preferredPeakBitRate to the desired limit, in bits per second, of the network bandwidth consumption for the given AVPlayerItem.
