We are pleased to announce that the Brightcove Player SDK for iOS v4.1.4 is
now available. You can download the zip file for the release here:

BCOVPlayerSDK:
https://github.com/brightcove/brightcove-player-sdk-ios/releases/tag/v4.1.4

This marks our first public release of version 4.1, a major version
increment that introduces new APIs for working with plugins. A list of the
changes is available in
CHANGELOG.md<https://github.com/brightcove/brightcove-player-sdk-ios/blob/2112c109205406c0976c34c69e50c3bf0111c537/CHANGELOG.md>
.

Along with v4.1.4, we are also happy to announce the initial public release
of the first two plugins for the Player SDK for iOS, integrating support
for Interactive Media Ads (IMA) using Google DFP server-side ad rules, and
content encryption using Widevine DRM and Brightcove Video Cloud. You can
download the zip files for these releases here:

BCOVIMA:
https://github.com/brightcove/brightcove-player-sdk-ios-ima/releases/tag/v1.0.4
BCOVWidevine:
https://github.com/brightcove/brightcove-player-sdk-ios-widevine/releases/tag/v1.0.4

The Brightcove-Player-SDK
v4.1.4<https://github.com/CocoaPods/Specs/blob/master/Brightcove-Player-SDK/4.1.4/Brightcove-Player-SDK.podspec>and
Brightcove-Player-SDK-IMA
v1.0.4<https://github.com/CocoaPods/Specs/blob/master/Brightcove-Player-SDK-IMA/1.0.4/Brightcove-Player-SDK-IMA.podspec>podspecs
have been merged into the public
CocoaPods/Specs <https://github.com/CocoaPods/Specs/> repo, so if you are
using CocoaPods, you can point your Podfile at the respective version
numbers of each spec. The Brightcove-Player-SDK-Widevine
v1.0.4<https://github.com/brightcove/BCOVSpecs/blob/master/Brightcove-Player-SDK-Widevine/1.0.4/Brightcove-Player-SDK-Widevine.podspec>podspec
is hosted in the public
Brightcove/BCOVSpecs <https://github.com/Brightcove/BCOVSpecs> repo, so you
can point your Podfile there. To do so, you may first have to add the
repository to your environment, using this command:

pod repo add BCOVSpecs git@github.com:brightcove/BCOVSpecs.git

We also have example code, demonstrating how to integrate these plugins
into the Player SDK for iOS in a sample app. (SampleBCOVIMAWidevine shows
how to use both plugins together in the same app):

SampleBCOVIMA: https://github.com/BrightcoveOS/SampleBCOVIMA
SampleBCOVIMAWidevine:
https://github.com/BrightcoveOS/SampleBCOVIMAWidevine
