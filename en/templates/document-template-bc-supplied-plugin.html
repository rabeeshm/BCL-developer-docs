Implement using Players module

To implement the NAME Plugin using the Players module, follow these steps:

  1. Open the Players module and locate the player.
  2. Click the link for the player to open the player properties.
  3. Locate the Plugins section and click Edit.
  4. The JavaScript for the plugin is provided and will not change for different player implementations. For the JavaScript URL, enter: 
     http:
  5. The CSS for the plugin is provided and will not change for different player implementations. For the CSS URL, enter:
     http:
  6. For the Name, enter PLUGIN NAME.
  7. Enter the plugin options. The example below will ...
    {
  "overlays": [
    {
      "align": "bottom",
      "start": 2
    }
  ],
  "content": "Default overlay content"
}
  8. Click Save and then Publish the player.
