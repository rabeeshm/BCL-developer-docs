In this topic, you will learn how to use a plugin to place a playlist selector icon in the controlbar. The selector permits selection of a playlist, then a video selection from the playlist. At player load, the first video in the first playlist is loaded and autoplayed.

The code for this plugin can be viewed and retrieved from the GitHub repo https://github.com/RevinKey/brightcove-multi-playlist.

The player sample

The following shows a player using the plugin to display three playlists from which a video can be selected. When you mouseover the player you will see this icon in the controlbar. When you mouse over that icon you will see the playlists the playlist is configured to display. You can then click on one of the playlists and select a video from the selected playlist's videos.

Using the plugin

The following sections explain how to use the plugin using the Players module and with code. In both cases, you will need to get the JavaScript and CSS files from the GitHub repo dist folder and save them in an Internet accessible location.

If you use this plugin, it will be your responsibility to periodically check the GitHub repo if you are interested in fixes or updates.

Implement using Players module

To implement this plugin using the Players module, follow these steps:

  1. Open the Players module and locate the player.
  2. Click the link for the player to open the player properties.
  3. Locate the Plugins section and click Edit.
  4. For the JavaScript URL enter the location where you saved the JavaScript code for the plugin.
  5. For the CSS URL enter the the location where you saved the CSS code for the plugin.
  6. For the Name, enter multiPlaylist.
  7. Enter the plugin options. The example below will display three playlists.
    {
  "playlists": [
    {
      "id": "4845949311001"
    },
    {
      "id": "1754200352001"
    },
    {
      "id": "1754200353001"
    }
  ]
}
  8. Check to be sure the Name, Options (JSON) section appears similar to the following, of course showing your playlist IDs.
  9. Click Save and then Publish the player.
