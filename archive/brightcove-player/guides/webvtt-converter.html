Converting Captions to WebVTT Format

The page you are looking for doesn't live here anymore -- redirecting you to in 3 seconds.

In this topic, you will learn how to convert DFXP captions for the Smart Player to WebVTT captions for the Brightcove Player. The workflow is explained, and also instructions on how to install and use a Python-based tool to convert your closed captions file from DFXP/TTML to WebVTT format. You can also convert from and to other valid captions file formats.

Also see the open source tool that converts your DFXP captions to WebVTT and adds the WebVTT captions to the video.

Workflow

Here's a general workflow for converting your old DFXP captions to the WebVTT format required for the Brightcove Player:

  1. Find all media with DXFP captions (using the Media API search_videos method, and including captioning in the fields to return).
  2. Get the DXFP files for each video (the URL for each DFXP file is included in the response for the search_videos request).
    Note: Each DFXP file should be in UTF-8 character encoding format.
  3. Convert using captions tool.
  4. If you've already added WebVTT captions to some of your videos, you won't want to overwrite those - you can identify which videos already have WebVTT captions by looking at the text_tracks object returned by the CMS API.
  5. Finally, for videos you need to add your converted captions files to, you can do this using the CMS API Update Video method.

Caption converter

This section introduces a Python-based converter than can be used to convert DFXP captions to the WebVTT format.

What you need

You will need the following software and files:

  • Git
  • Python 2.7
  • Your DFXP/TTML captions file (in UTF-8 character encoding format)
  • The GitHub repository for pycaption
  • The GitHub repository for pycaption-cli

Even though the later versions of Mac MacOS come with Python, this version of Python is great for learning, but not so good for development. So, you might want to install Python manually.

Supported formats

You can convert captions files between these file formats:

  • Read: DFXP/TTML, SAMI, SCC, SRT, WebVTT
  • Write: DFXP/TTML, SAMI, Transcript, SRT, WebVTT

Installation

Follow these steps to install the converter tool:

  1. If you don't already have it, install Git on your computer. Refer to the Git downloads page.

    You can use the following command to check the current version installed on your computer:

    git --version
  2. If you don't already have it, install Python 2.7 on your computer. Refer to the Python downloads page.

    You can use the following command to check the current version installed on your computer:

    python --version
  3. Clone the GitHub repository for pycaption to your desktop. This module reads a captions file and writes out a captions file in your desired format.
    Note: It has been found you need to make an edit to the code in this repo. Once you have cloned the repo, move into pycaption/dfxp folder. Then in the base.py file, edit line 62. Change it from

    to

    .
  4. Clone the GitHub repository for pycaption-cli to your desktop. This module adds a command line interface for the pycaption module.
  5. At the command line, change the directory to the folder where you cloned the pycaption repository.

  6. To install this Python module on a Mac, enter the following :

    python setup.py install

    On a Mac, if you get this error: Could not find function xmlCheckVersion in library libxml2. Is libxml2 installed?, you may need to install or upgrade the command line tool for Xcode. Try the entering the following command:

    xcode-select --install

    Now, you should be able to enter the original Python install command.

    For Windows, use:

    setup.py install
  7. At the command line, change the directory to the folder where you cloned the pycaption-cli repository.

  8. You will update the command line interface module to read and write WebVTT files. On your computer, locate the pycaption-cli/pycapcli/caption_converter.py file and replace it with this code:

    You can also download the new caption_converter.py file with WebVTT code.
    import optparse
import codecs

import pycaption


def main():
    parser = optparse.OptionParser("usage: %prog [options]")
    parser.add_option("--sami",
            action='store_true',
            dest='sami',
            help="write captions in SAMI format",
            default=False,)
    parser.add_option("--dfxp",
            action='store_true',
            dest='dfxp',
            help="write captions in DFXP format",
            default=False,)
    parser.add_option("--srt",
            action='store_true',
            dest='srt',
            help="write captions in SRT format",
            default=False,)
    parser.add_option("--webvtt",
            action='store_true',
            dest='webvtt',
            help="write captions in webvtt format",
            default=False,)
    parser.add_option("--transcript",
            action='store_true',
            dest='transcript',
            help="write transcript for captions",
            default=False,)
    parser.add_option("--scc_lang",
            dest='lang',
            help="choose override language for input",
            default='',)
    parser.add_option("--scc_offset",
            dest='offset',
            help="choose offset for SCC file; measured in seconds",
            default=0)
    (options, args) = parser.parse_args()

    try:
        filename = args[0]
    except:
        raise Exception(
        ('Expected usage: python caption_converter.py <path to caption file> ',
        '[--sami --dfxp --srt --webvtt --transcript]'))

    try:
        captions = codecs.open(filename, encoding='utf-8', mode='r').read()
    except:
        captions = open(filename, 'r').read()
        captions = unicode(captions, errors='replace')

    content = read_captions(captions, options)
    write_captions(content, options)


def read_captions(captions, options):
    scc_reader = pycaption.SCCReader()
    srt_reader = pycaption.SRTReader()
    sami_reader = pycaption.SAMIReader()
    dfxp_reader = pycaption.DFXPReader()
    webvtt_reader = pycaption.WebVTTReader()

    if scc_reader.detect(captions):
        if options.lang:
            return scc_reader.read(captions, lang=options.lang,
                                   offset=int(options.offset))
        else:
            return scc_reader.read(captions, offset=int(options.offset))
    elif srt_reader.detect(captions):
        return srt_reader.read(captions)
    elif sami_reader.detect(captions):
        return sami_reader.read(captions)
    elif dfxp_reader.detect(captions):
        return dfxp_reader.read(captions)
    elif webvtt_reader.detect(captions):
        return webvtt_reader.read(captions)
    else:
        raise Exception('No caption format detected :(')


def write_captions(content, options):
    if options.sami:
        print pycaption.SAMIWriter().write(content).encode("utf-8")
    if options.dfxp:
        print pycaption.DFXPWriter().write(content).encode("utf-8")
    if options.webvtt:
        print pycaption.WebVTTWriter().write(content).encode("utf-8")
    if options.srt:
        print pycaption.SRTWriter().write(content).encode("utf-8")
    if options.transcript:
        print pycaption.TranscriptWriter().write(content).encode("utf-8")


if __name__ == '__main__':
    main()
  9. To install this Python module, enter the same command you used in step 6.

Convert files

Now, you are ready to convert captions files from one format to another.

  1. At the command line, return to the folder where you cloned the pycaption-cli repository.

  2. On your computer, locate your captions file. In this example, we are converting a DFXP/TTML file to WebVTT format. Here is a sample DFXP/TTML file:
