Dimensions: videocloud

The videocloud domain collects data for the following dimensions:

Dimension

Explanation

account

passed as account to the Data Collection API (should be a Video Cloud account id).

player

passed as player to the Data Collection API (should be a Video Cloud player id in the specified account).

video

passed as video to the Data Collection API (should be a Video Cloud video id in the specified account).

referrer_domain

Top private domain derived from the source parameter passed to the Data Collection API (source should be the referrer for the page where events are collected).

source_type

Source type that resulted in traffic. Derived from the source parameter passed to the Data Collection API (source should be the referrer for the page where events are collected).

search_terms

Search terms that resulted in traffic. Derived from the source parameter passed to the Data Collection API (source should be the referrer for the page where events are collected).

device_type

device types for the user's device. Derived from the user=agent parameter passed to the Data Collection API.

device_os

device operating system for the device of the user. Derived from the user=agent parameter passed to the Data Collection API.
