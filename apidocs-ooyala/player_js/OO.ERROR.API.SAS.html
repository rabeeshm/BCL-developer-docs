Namespace: SAS

OO.ERROR.API. SAS

Represents the OO.ERROR.API.SAS Ooyala V3 Player Errors for the Stream Authorization Server. Use message bus events to handle errors by subscribing to or intercepting the OO.EVENTS.ERROR event. For more information, see Errors and Error Handling Overview.
