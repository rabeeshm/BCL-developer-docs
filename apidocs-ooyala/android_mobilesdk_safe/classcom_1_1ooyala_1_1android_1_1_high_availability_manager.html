Detailed Description

This manages the HighAvailability (HA) logic to retry to load a video when it fails to play.

This will be used if you have HA enabled in your Ooyala Backlot account. In Backlot you need to add a metadata field to each video that you want to activate HA on, the metadata field in Backlot should be called "ha_enabled", and it accepts a boolean value (either true or false). When set to true and the asset fails to play because of an error, HA will try to reload it again. You can also specify an additional metadata field called "ha_count" to specify the number of times we HA is going to retry to load a video.
