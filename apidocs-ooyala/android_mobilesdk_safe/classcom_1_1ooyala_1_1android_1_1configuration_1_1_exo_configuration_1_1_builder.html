Builder com.ooyala.android.configuration.ExoConfiguration.Builder.setFilterHdContent ( boolean  filterHdContent)

Set this flag to "false" if you wanted to play HD contents on devices which are running widevine L3 security.

Most widevine L3 security devices are running on Android 4.4.X OR api 19.

CAUTION : We do not recommend using this. This is only for customers who want to have HD contents all the time. If customers are having low bandwidth connections. It might cause streaming performance problems as it will prioritize HD contents rather than best stream according to network connections speed.
