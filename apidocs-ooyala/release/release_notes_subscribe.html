Subscribe to Ooyala Release Notes

You can subscribe to Ooyala release notes to receive automated notifications whenever new release notes are published.

Subscribe to Release Notes with the Ooyala Status Dashboard

Subscribing to release notes from the Ooyala Status Dashboard will send you email or text notifications for all Ooyala product release notes.
  1. Go to http://status.ooyala.com.
  2. Click Subscribe to Ooyala in the upper right corner of the page.
  3. In the Components menu, scroll to the bottom of the list and select Release Notes.
  4. To receive email notifications whenever new release notes are published, select E-mail and enter your email address.
  5. To receive text notifications whenever new release notes are published, select Text Message and enter your email address.
  6. Click Subscribe.

Subscribe to Release Notes with the Ooyala Community

Subscribing to release notes from the Ooyala Community will send you email notifications for the release notes for the Ooyala Video Platform Products and/or Ad Products.

  1. For Video Platform release notes, go here.

    OR

    For Ad Product release notes, go here.

  2. Click Topic Options > Subscribe.
