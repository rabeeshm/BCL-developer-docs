Detailed Description

This class is used to hold a VAST 3.0 time offset value.

Supported format:

  1. Seconds in HH:MM:SS.MMM, e.g. 01:30:20.250
  2. Percentage end up with a %, e.g. 20%
  3. Predefined position starting with #, e.g. #1
